🚀 アプリの3大特徴

  • 忘却曲線アルゴリズム: 記憶が薄れる最適なタイミングで「復習」を自動出題します。
  • 進捗の完全可視化: 下記ダッシュボードで本日のミッションや連続学習日数を把握できます。
  • 動的UI生成: data.txt の内容に応じ、学習カテゴリの選択肢が全自動で生成されます。

📂 data.txt の取り扱いについて

  • フォーマット: 1行につき1つの表現。項目は必ず「タブ（Tab）」で区切り、以下の7項目を記述してください。
    ターゲット表現 / 発音記号（ない場合は -） / 和訳 / 例文 / 例文和訳 / 解説 / カテゴリ
  • カテゴリの自由追加: 7項目目の「カテゴリ」に新しい名前（例：会計学、出張用など）を記述するだけで、アプリ側に自動で新しいカテゴリとして認識・表示されます。
  • データの更新と履歴保持: data.txt の和訳やカテゴリを修正・追加しても、「いつ学習して次はいつ復習するか」という学習履歴は自動で引き継がれます。また、不要な行をテキストから削除すれば、アプリ内からも幽霊データとして残ることなく綺麗に消去されます。

💡 一流の学習ルーティン（推奨される使い方）

  1. (1) Podcastインプット: 移動中などにCNNを聴き、気になる表現を即座にストックします。
  2. (2) リプロダクション（耳慣らし）: ランダムに英語の音声を聴き、口に出すトレーニングを「10回」行います。
  3. (3) 新規単語の習得（インプット）: 未学習の単語を1日「20件」、登録順に学習します。
  4. (4) 本日の復習（記憶の定着）: 忘却曲線が今日と判断した単語だけが出題されます。ここを「残り 0 件」にすることが目標です。
